What is YouTube Web?

YouTube Web is a Graph Visualization program that connects youtubers by their recommended and featured channels. The web application also includes a side bar that allows you to search up a youtuber to see more information (Subscribers, Views, Videos, and Comments). The visualization uses Youtube's API to obtain data, Alchemy.js (a graph drawing application) to populate and create the graph's nodes and edges, and HTML/CSS and Javascript to give a user a highly interactive experience. Current features include: Interacting with the graph, viewing more information about a user, and searching for users. In the future, we hope to add VR/AR to make the experience even more immersive!
